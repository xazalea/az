# Azalea VS Code Extension

The official Visual Studio Code extension for **Azalea**, the AI-Native Programming Language.

## Features

*   **Syntax Highlighting**: Full support for Azalea's indentation-based syntax and keywords (`ai`, `agent`, `rag`, `macro`).
*   **Theme**: Includes the "Azalea Sapphire Mist" color theme for a professional, eye-pleasing experience.
*   **Snippets**: Productive snippets for common patterns like `agent` definitions and `import ai`.

## Usage

1.  Open any `.az` file.
2.  Enjoy the "Sapphire Mist" theme and syntax highlighting.
3.  Use the command palette or just type to start coding.

## Requirements

*   Visual Studio Code 1.80.0 or higher.

## Extension Settings

This extension contributes the following settings:

*   `azalea.enableAI`: Enable/disable AI features in the editor (future).
